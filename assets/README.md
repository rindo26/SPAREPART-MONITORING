Simpan logo PNG yang Anda lampirkan sebagai `assets/logo.png` untuk digunakan oleh halaman.

Catatan perubahan saat ini:

- `index.html` sekarang menunjuk ke `assets/logo.png` (jika file ada, itu akan tampil di navbar).
- Ada juga placeholder `assets/logo.svg` yang dapat Anda hapus jika sudah mengganti dengan PNG.

Cara memasang logo Anda:

1. Unduh lampiran PNG yang Anda kirim di percakapan ini.
2. Simpan file tersebut dengan nama `logo.png` di folder `assets/` (jadi path lengkap: `assets/logo.png`).
3. Muat ulang browser untuk melihat logo baru.

Jika Anda ingin saya menambahkan logo PNG secara langsung ke repo, unggah file PNG melalui mekanisme unggah file di VS Code atau beri tahu saya dan saya akan membantu langkah-langkahnya.
